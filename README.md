# DilSeGive

DilSeGive is a donation and crowdfunding platform that connects donors with people in need.

## Features
- User Authentication
- Create Donation Campaigns
- Secure Donations
- Campaign Tracking
- Admin Dashboard

## Tech Stack
Frontend:
- React.js
- TypeScript
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Installation

1. Clone the repository
git clone <repository-url>

2. Install dependencies
npm install

3. Start the application
npm start
